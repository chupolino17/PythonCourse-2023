# Как сдать курс

## Активности:
1. Контесты (5)
    - 10 баллов за каждый
    - 2 недели на контест
2. Interactive (2)
    - 20 баллов (ранжируются по результату)
2. Web Code Review (1)
    - Работа над кодом в **несколько** итераций
    - Есть обяязательная часть
    - Доп баллы (до 2ух) к финальной оценке за бонусную
    - Условия позже
    - Блокирующее
3. Лабораторная работа (1):
    - 30 баллов
    - Блокирующее
4. TEST (1)
    - 20 баллов
    - Блокирующее
5. Final CONSTEST (1)
    - 50 баллов
